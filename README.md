# fastswitch

[Download from github](https://github.com/jvr-ks/fastswitch/raw/master/fastswitch.exe)  
Virus check see below. 
  
Supers simple app (Windows only) to switch between app-windows via hotkeys.  
  
**In "cmdlinedev" > version 0.128 this functionality is already built in!**

Hotkey-file: "cldfastswitch.txt"  
(identical with the Hotkey-file from "cmdlinedev")  
  
Format is: hotkey, app-title  
  
hotkey is a hotkey-definition, see [Autohotkey Hotkeys](https://www.autohotkey.com/docs/Hotkeys.htm)   
app-title is any part of the window-title


Already defined in the Hotkey-file:  
* Notepad++ with \[ALT] + \[n], i.e. "!n"
* Dopus with \[ALT] + \[d], i.e. "!d" 
  
To stop the app start: fastswitch.exe remove

  
##### Executable
* "fastswitch.exe"  
  
##### Sourcecode: [Autohotkey format](https://www.autohotkey.com)
* "fastswitch.ahk".  
  
##### Requirements
* Windows 10 or later only.  
* Portable app, nothing to install.
